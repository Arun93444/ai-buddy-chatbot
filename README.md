# 🤖 AI Chatbot - Buddy

A conversational AI chatbot built with Python and Google Gemini API.

## What it does
- Answers any question using real AI
- Remembers the full conversation history
- Has a friendly personality as an AI mentor
- Runs completely from the command line

## Skills used
- Python
- Google Gemini API
- REST API calls
- Functions, Loops, Dictionaries

## How to run
1. Clone this repository
2. Install requirements: pip install requests python-dotenv
3. Add your Gemini API key to .env file
4. Run: python my_chatbot.py
