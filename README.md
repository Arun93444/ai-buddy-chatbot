## 📌 About
Buddy AI is a conversational AI chatbot built with Python and Google Gemini API. It remembers your entire conversation, has a friendly personality, and saves all chats to MongoDB.

##  Features
-  Context-aware AI responses using Google Gemini
-  Full conversation memory across sessions
-  MongoDB Atlas for permanent chat storage
-  Beautiful dark UI built with Streamlit
-  Deployed live — accessible from anywhere

##  Tech Stack
| Technology | Purpose |
|-----------|---------|
| Python | Backend logic |
| Google Gemini API | AI responses |
| MongoDB Atlas | Chat storage |
| Streamlit | Web interface |
| REST API | API communication |
